# ShadowImageView
## Abstract 摘要
    更加细腻的阴影效果，在一些提高用户体验的特殊场景使用

## Usage 使用方法

### In layout

```
	    <com.yinglan.shadowimageview.ShadowImageView
                android:layout_width="300dp"
                android:layout_height="300dp"
                app:shadowRound="20dp"
                app:shadowSrc="@mipmap/lotus" />

```
